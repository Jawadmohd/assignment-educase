# PropX Web Application

## Overview

This project is a simple authentication web application built using React. The application provides a landing page where users can choose between **Login** and **Signup**, enter their details, and get authenticated.

The project demonstrates modern React development practices including reusable components, form validation, and global state management.

---

## Features

* Landing/Home page
* User Signup form
* User Login form
* Form validation
* Authentication flow
* Dynamic and reusable form fields
* Responsive and clean UI
* State management using React Context API

---

## Technologies Used

### Frontend

* React
* React Hook Form
* React Context API
* CSS

---

## Authentication Flow

1. User opens the landing page.
2. User selects Login or Signup.
3. User enters required details.
4. Form validation is performed using React Hook Form.
5. User data is managed through React Context.
6. Authentication state is updated.

---

## Form Validation

The project uses **React Hook Form** for:

* Required field validation
* Email format validation
* Password validation
* Controlled form submission

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```plaintext
http://localhost:5173
```

---

## Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
